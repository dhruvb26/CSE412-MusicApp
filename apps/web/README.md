# Web

Next.js frontend for the music database. Displays a grid of albums, artists, and songs with search and a detail panel.

## Setup

```bash
bun install   # from repo root
bun dev       # from this directory, or `bunx turbo dev` from root
```

Runs at [http://localhost:3000](http://localhost:3000). Requires the API to be running at `http://localhost:8000`.
